# Habit Tracker MVP

A simple habit tracking application built with a Python backend and a basic HTML/CSS/JS frontend.

The goal of this project is to provide a minimal but functional MVP where users can:
- Register and log in
- Create and delete habits
- Mark habits as done for the current day
- View a simple weekly summary

## Technologies
- Python (Flask)
- HTML / CSS / JavaScript
- JSON file storage
